# DebtConnect
This is a web based application that helps people with debt to openly discuss their concern, make connections and to have a safe place to find a consultant to talk to.

To run the app:
1. Download zip file
2. Open VS code
3. Go to terminal
4. Type "cd debt-app"
5. Type "npm start"
6. Click Log-in button to start
7. Happy explore!
